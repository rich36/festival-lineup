Festival Lineup

Rich Reuter
rich@richreuter.com

Festival Lineup is a configurable HTML 5 application designed to provide a way to view the lineup of a music festival using a smartphone.

FEATURES
* Automatically updates to show who is currently playing
* Supports multiple stages
* Social Network Sharing
* Configurable content. Includes variable content for before the festival, after the festival and when no one is playing
* Provides the ability to easily configure the artists, stages and performance times
* Self-contained. Built on HTML/CSS/Javascript.  No database or server configuration required.
* Provides the ability to artificially set the time to test what the site and current lineup look like at specific times.
